# Cheat Sheet

`[ფუძე]` = ます ფორმაში ჩასმა და ます-ს მოშორება

`ない` - აწმყო უარყოფითი შემოკლებითი ფორმა

`た` - წარსული დადებითი შემოკლებითი ფორმა

`〜ている` - განგრძობითი დრო

`〜んです` - ახსნა, ემოციის ჩადება (მაგ: `しずかなんです`, `かわいいんです`)

`〜のです` - წერილობითში იგივე რაც `〜んです`


## შედარებები

`A のほおが B より すきです` - ორი რამის შედარება

`A と B と どちら のほおが C ですか` - ორი რამის შედარება

`A と B と C の なかで なにが C ですか` - ორზე მეტი რაღაცის შედარება


#


`〜て 〜て ` - მოქმედებების გადაბმა (სრული) (მხოლოდ ბოლო ზმნა ჯდება საჭირო დროის ფორმაში)

`〜たり 〜たりする` - მოქმედებების გადაბმა (არასრული, და ა.შ.) (მხოლოდ ბოლო ზმნა ჯდება საჭირო ფორმაში)

#

`。。。と。。。と。。。` - არსებითების გადაბმა, სრული ჩამონათვალი

`。。。や。。。や。。。` - არსებითების გადაბმა, არასრული ჩამონათვალი (და ა.შ.)


#

`ことができる` - შეძლება, იგივე რაც ზმნის პოტენციური ფორმა (მაგ: `のむことができる`)

`〜たほおがいいです` - უმჯობესია

`〜ないほおがいいです` - უმჯობესია არ

`〜ないでください` - თხოვო თავის შეკავება მოქმედებისგან

# 

`〜なければなりません` - ამის გარეშე არ გამოვა (must) (მაგ: `べんきおしなければなりません`)

`〜なければいけません` - ამის გარეშე არ გამოვა (must)

`〜なくちゃいけません` - ამის გარეშე არ გამოვა (must)

#

`[ზმნა აწმყო შემოკლებითში]+つもりです` - ვაპირებ (მაგ: のむつもりです、のむつもりだ)

#

`[შემოკლებითი ფორმა]+とおもいます` - ვფიქრობ (მაგ: `Xは かわいい とおもいます`)

`[შემოკლებითი ფორმა]+と言っていました` - თქვა, ციტირება

`[წარსული შემოკლებითი ფორმა]〜ことがある` - მოქმედება გაგიკეთებია (have done)

`[აწმყო შემოკლებითი ფორმა]〜ことがある` - მოქმედებას ვაკეთებ ხოლმე

# 

`[ფუძე]〜ましょう` - შეთავაზება (მაგ: ერთად დავლიოთ - `のみましょう`)

`[ფუძე]〜ましょうか` - კითხვითი შეთავაზება, ასევე დახმარების შეთავაზებაც (მაგ: ხომ არ გვეჭამა - `たべましょうか`)

`[ფუძე]〜たい` - მინდა (მაგ: `のみたい`)

`[ფუძე]〜たがっている` - (სხვას) უნდა (მაგ: `のみたがっている`)

`[ფუძე]〜すぎる` - ზედმეტად

`[ფუძე]〜にくい` - რთულია (მაგ: `ビールおのみにくいです`)

`[ფუძე]〜やすい` - მარტივია (მაგ: `おさけおのみやすいです`)

`[ფუძე]〜になる` - გახდა (თავისით) （მაგ: `先生になる`)

`[ფუძე]〜くなる` - გახდა (თავისით) （მაგ: `かわいくなる`)

`[ფუძე]〜にする` - გახდა (ჩემი ზემოქმედებით)

`[ფუძე]〜くする` - გახდა (ჩემი ზემოქმედებით)

`[ფუძე]〜に[ზმნა]` - მაგ. საჭმელად მიდის - `たべにいきます`

# 

`〜てください` - გთხოვთ

`〜てもいいです` - ნებართვის გაცემა

`〜てもいいですか` - ნებართვის აღების თხოვნა

`〜てはいけません` - ნებართვის არ დართვა

`〜てみる` - ვცდი

`〜てみたい` - ვისურვებდი

`〜てあがる` - მოქმედების გაცემა (მაგ: `よんであがる` - წავუკითხავ)

`〜てもらう` - მოქმედების მიღება (მინიშნებების გაცემით) (მაგ: `よんでもらう` - წამიკითხავს)

`〜てくれる` - მოქმედების მიღება (თავისით) (გამცემს: `は`, მიმღები: მე) (მაგ: `よんでくれる` - წამიკითხავს)

`〜ていただく` - けいご-ში იგივე რაც もらう.

`[შემოკლებითი ფორმა]〜でしょう` - ალბათ

#

`〜たらいいですか` - ორჭოფობა (მაგ: `のんだらいいですか` - რომ დავლიო?)

#

`なん`　- რა

`なに`　- რა

`なにか＿`　- რაღაც, რამე, რაიმე

`なに＿も`　- არაფერი

`だれ`　- ვინ

`だれか＿`　- ვიღაცა

`だれ＿も`　- არავინ

`どこ`　- სად

`どこか＿`　- სადღაც

`どこ＿も`　- არსად

შესაძლებელია `＿`-ს ადგილას ჩაჯდეს `に、へ、で`

#

`これ` - ეს

`それ` - ეგ

`あれ` - ის

`どれ` - რომელი ერთი (が)

`この。。。` - ეს　。。。

`その。。。` - ეგ　。。。

`あの。。。` - ის　。。。

`どの。。。` - რომელი 。。。(が)

#

`[ზმნის შემოკლებითი]〜のが...` - მაგ. კითხვა მიყვარს - のむのがすきです

`[ზმნის შემოკლებითი][არსებითი]` - ზმნით არსებითის განსაზღვრა (მაგ: `本の読む人`-წიგნის მკითხველი ადამიანი)

`。。。まだ。。。〜ていません` - ჯერ (ჯერობით) არ [მოქმედება]

`。。。もう。。。〜ました` - უკვე [მოქმედება]

`から` - რადგან, იმიტომ რომ (როგორც შუაში, ისე ბოლოში)

`ので` - იგივეა რაც `から`, ზრდილობიანი ფორმა, გამოიყენება შემოკლებითთან

`の` - ტავტოლოგიის თავიდან ასაცილებლადაც

`で` - რაღაცით, (მაგ. კალამით ვწერ: `ペンで書きます`)

`。。。が、。。。` - მაგრამ　（წინადადების შუაში)

`は ー＞ が、お` - უარყოფით წინადადებებში ხშირად `は` ანაცვლებს `が、お`-ს

`。。。ませんか` - ხომ არ ... ?

`[ზედსართავის ფუძე]〜そうです` - როგორც ჩანს ... (გამონაკლისი `いい` - `よさそうです`)

`[ზედსართავის ფუძე]〜なさそうです` - როგორც ჩანს　არ ... (ない-ში გადაგვყავს და なさ-თი ვანაცვლებთ)

`なら` - გამოიყენება `は、が`-ს მსგავსად გამოიყენება შედარება/დაპირისპირებისთვის (აგდებს `は、が、も`-ს)

`たかい・たかく` - ძვირი / ძვირად

`へたな・へたに` - მოუხერხებელი / მოუხერხებლად
